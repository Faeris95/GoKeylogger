# GoKeylogger [![License](https://img.shields.io/badge/licence-GPLv3.0-green.svg](https://raw.githubusercontent.com/Faeris95/GoKeylogger/master/LICENSE?token=AQYjCTVzZPDeUfD7VWrNhzERQAM-rMYZks5Xle7twA%3D%3D)

GoKeylogger is an advanced Windows Keylogger that works only with AZERTY keyboards layout
With other layouts all keys except [a-Z][Pad 0-9] may not work properly
